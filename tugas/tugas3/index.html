WEB BROWSER

Definisi Web Browser
Web Browser adalah suatu perangkat lunak yang digunakan untuk menampilkan halaman-halaman web yang ada

Adapun istilah-istilah yang sering muncul pada saat kita menggunakan web browser adalah sebagai berikut:

  1. web : halaman-halaman web yang saling terhubung dalam suatu website
  2. Hompage : Halaman awal ketika situs dimunculkan
  3. URL : Alamat unik pada suatu halaman web
  4. www : Kumpulan dari dokumen-dokumen elektronik
  5. Portal : Website yang menyediakan berbagai jenis layanan

3 Jenis web browser yang sering dipakai:

  • Internet Explorer
  • Netscape
  • Mozilla
