HTML (Hypertext Markup Language):
Adalah sebuah protokol yang digunakan untuk membuat format suatu dokumen web yang mampu dibaca dalam Browser dari berbagai platform komputer. Sifat bahasa HTMLini adalah client script , dimana dokumen tersebut dapat dibuka dalam komputer stand alone yang tidak membutuhkan server untuk dapat menampilkannya di dalam browser. Dokumen HTML merupakan file yang pada umumnya berekstensi .htm atau .html, dimana bahasa HTML tersebut tersusun atas tag yang berformat "<>"
